# HangmanGame
Hangman game on Python with hangman drawn ASCII 
## Info
Words located at data\words.txt. Put your words there, one line - one word, it must be blank line at end of file
## Example of hangman picture
```
Attemps remain 0
            _______
            |/
            |     ( )
            |     _|_
            |    / | \
            |     / \
            |
          __|________
          |         |
You lose!
```
